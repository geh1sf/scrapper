# Kraimorie Daily Property Tracker

A simple, effective approach to monitor **ALL** new property listings in Kraimorie, Burgas daily.

## What This Does

- Tracks ALL property types in Kraimorie (apartments, houses, commercial, land)
- Shows new listings each day - never miss anything
- Highlights Argosdom properties - special attention to preferred agency  
- Creates daily website with all findings
- Runs automatically via GitHub Actions every day at 8 AM UTC

## Benefits

- Never miss anything - see all market activity
- Argosdom priority - highlighted when available
- Daily automation - zero manual work
- Clean website - easy to review findings

This approach ensures you catch every opportunity in Kraimorie while keeping special focus on your preferred Argosdom agency.
